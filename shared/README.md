# shared

Эта директория зарезервирована под общие типы, transport-контракты, event names и базовые доменные модели, которые должны использоваться и клиентом, и сервером.

Ближайшая задача для этого слоя:

- вынести общие socket event names
- зафиксировать общие DTO и payload contracts
- убрать дублирование строковых литералов между `client` и `server`

Первый шаг уже сделан:

- [socket-events.ts](socket-events.ts) — канонический список socket event names для постепенного выравнивания client/server кода
